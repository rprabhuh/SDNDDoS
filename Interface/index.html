SDN Enabled Big Data Analytics for DDoS Resistant Networks

Abstract

DDoS (Distributed Denial of Service) is becoming a major security threat on the Internet, and real-time DDoS detection and mitigation can be a challenging task. We propose a real-time DDoS packet classification system in the SDN (Software Defined Network) context to address the problem with a learning-based approach using Apache Spark computation engine.We define the features of DDoS flows and provide a real-time response learning approach which is alternative to existing packet filtering based systems.
